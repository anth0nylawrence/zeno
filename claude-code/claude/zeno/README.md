# Zeno Runtime Artifacts

This directory stores Claude Code Zeno artifacts generated by hooks.

Subfolders:
- state/: per-session state snapshots
- evidence/: evidence ledger JSONL
- claims/: claim ledger JSONL
- snapshots/: per-turn snapshots and pre-compact snapshots
- cursors/: transcript cursor offsets
- tool_events/: tool output/event logs
- notify.log: per-turn checkpoint log
